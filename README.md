# SWA_Assignment_1
 Weather forecast project

 ## Development requirements:
 - Run
```npm install --save-dev webpack webpack-cli ```
```npm install --save-dev babel-loader @babel/core @babel/preset-env```
- Running the webpack:
    - Dev: ``` npm run build```
    - Prod: ``` npm run prod```
